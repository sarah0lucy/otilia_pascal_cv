Bachelor’s Thesis: Web Application for Renting Clothes for Special Events

This thesis presents the development of a sustainable digital platform for renting clothing, footwear, and accessories, as a response to the environmental challenges generated by the fashion industry. The project offers a responsible and economical alternative to overconsumption, particularly around special events, by encouraging reuse and reducing unnecessary purchases, as well as the excessive use of raw materials and chemical substances commonly used in textile production.
The application was built using PHP and MySQL for the back-end, and HTML, CSS, JavaScript, and Bootstrap for the front-end, with a strong focus on accessibility, intuitive navigation, and operational efficiency.
Core features include user registration, password recovery, product filtering by preference, the ability to mark items as favorites, shopping cart management, order processing, temporary rentals, return requests (including late return fees), and an admin panel for catalog and return management. These functionalities ensure a transparent, user-friendly, and automated experience.
Overall, the project demonstrates how digital technology can contribute to solving real-world issues by supporting a more sustainable consumption model that aligns with both economic user needs and the imperative of environmental protection.
